Built for one operator's own workflow — then shared.

Why Matrix exists

ZB Automations builds tools for running an automation-heavy workflow without handing data to a dozen different SaaS vendors. Matrix started as a personal dashboard — chat across AI providers, an on-disk IDE, autonomous agents, notes, email, calendar — consolidated into one self-hosted control plane instead of a browser full of separate tabs and subscriptions.

What we build

Matrix is the dashboard itself: multi-provider AI chat, autonomous agents with tool access (GitHub, Slack, Gmail, Drive, Calendar), an FTS5-backed memory bank, a deep research engine, and a full on-disk IDE (VS Code via code-server) embedded in the browser — all backed by a local, encrypted SQLite database.

Matrix Builder is the integrated AI site and dashboard generator built into Matrix — describe a site once, get an agency-grade build with real type systems and motion, not a generic template.

Self-hosted by design

Everything runs on infrastructure you control. There's no ZB Automations-operated backend collecting your chat history, notes, or credentials — API keys and secrets are encrypted at rest (AES-256-GCM) on your own server, and the only network calls Matrix makes are the ones you configure to your own chosen AI providers and integrations.
