Privacy Policy

Last updated 07 July 2026

What this covers

This policy covers two separate things: the public marketing site at zbautomations.ie, and the Matrix dashboard software itself. They're handled very differently, so we split them out below rather than writing one policy that blurs the two.

The zbautomations.ie website

This is a static site with no accounts, no forms, no cookies, and no analytics or tracking scripts of any kind. Your web server logs standard request metadata (IP address, user agent, requested path, timestamp) — this is generic infrastructure logging, not used for profiling or shared with any third party for advertising.

The Matrix dashboard

Matrix is self-hosted software: ZB Automations does not operate a hosted version of Matrix, does not receive your data, and cannot access an instance you run yourself. Whatever you enter into your own deployment — chat messages, notes, memory bank entries, connected accounts, API keys — is encrypted at rest (AES-256-GCM) and stored in a local SQLite database on infrastructure you control. Whether that data ever leaves your machine is entirely up to you and whichever third-party AI providers or integrations you choose to connect (e.g. Anthropic, OpenAI, Google, GitHub, Slack) — those providers' own privacy policies govern data you send them, not this one.

Third-party services

If you connect Matrix to a third-party AI provider or integration, the requests you make are sent directly from your own server to that provider, under whatever account and API key you supplied. ZB Automations is not a party to that data flow and has no visibility into it.

Changes to this policy

If this policy changes, the "last updated" date above will change too. There's no mailing list or account to notify since there's nothing to sign up for.

Contact

Questions about this policy can be raised via the GitHub repository.
