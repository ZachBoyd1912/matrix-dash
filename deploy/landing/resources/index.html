Frequently asked questions

Common questions about Matrix, answered directly.

Is Matrix a hosted product I can sign up for?

No. There's no public signup, free trial, or multi-tenant SaaS offering. Matrix is self-hosted software — you deploy it on your own server (a $0-tier cloud VM is enough) and run it yourself.

Which AI providers does Matrix support?

20+ providers out of the box, including Anthropic, OpenAI, Google Gemini, Mistral, xAI Grok, DeepSeek, Groq, OpenRouter, and local models via Ollama or LM Studio. You bring your own API keys; Matrix stores them encrypted (AES-256-GCM) on your own server.

Where does my data actually live?

In a local SQLite database on whatever server you deploy Matrix to. Chat history, notes, memory bank entries, and connected-account tokens never pass through any ZB Automations-operated backend, because there isn't one.

What's the difference between Matrix and Matrix Builder?

Matrix is the dashboard — chat, agents, IDE, notes, email, calendar, memory. Matrix Builder is a feature inside Matrix: an AI site/dashboard generator that produces agency-grade builds from a single prompt, editable live without leaving the dashboard.

What does it cost to run?

The reference deployment runs on a free-tier (e2-micro) Google Cloud VM behind Caddy for automatic HTTPS — infrastructure cost is effectively €0/month beyond whatever AI provider usage you generate directly with your own API keys.

Is the source code available?

Yes — see the GitHub repository for source, setup scripts, and the current roadmap.
