Matrix Dash

AI command center. Local-first. Runs on your machine.

Privacy Policy

Last updated: June 2026

Matrix Dash is a local-first application. All data is stored on your local machine in ~/MatrixDash/. No data is transmitted to external servers unless you explicitly connect third-party integrations (GitHub, Slack, Google). API keys and OAuth tokens are encrypted at rest using AES-256-GCM with a machine-local key. The application does not use analytics, tracking, or telemetry of any kind.
